What is Float?

Float is a CSS positioning property which takes an element from its normal flow and positions it along the left or right side of its container. The text and other inline elements will then wrap around it.

What is inline-block?

An element set to inline-block just like inline, will display elements next to each other with the natural flow of the text but then also allows you to control the width and height. Example of elements with the display type of inline-block in the image tag

Applying Inline-blocks

Values of Float

Left: The element generates a block box that is floated to the left. Content flows on the right side of the box, starting at the top.

Right: Similar to 'left', except the box is floated to the right, and content flows on the left side of the box.

None: The box is not floated.

Inherit: Takes the same specified value as the property for the element's parent.
