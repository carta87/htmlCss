Errores comunes de CSS

A continuación, se muestra una lista de errores comunes de CSS:

  • Nombres de etiquetas mal escritos o incorrectos
  • Nombres de propiedades mal escritos o incorrectos
  • Valores de propiedades mal escritos o incorrectos
  • Falta de los dos puntos (:) después del nombre de una propiedad
  • Falta del punto y coma (;) después de valores de propiedad
  • Falta de la coma entre nombres de etiquetas (,)
  • Falta {} alrededor de las propiedades
