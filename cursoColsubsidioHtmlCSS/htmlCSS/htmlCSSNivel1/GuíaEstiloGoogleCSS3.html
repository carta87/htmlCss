Google Style Guide

The Google style guide helps you write code that is easy to read.

CSS Formatting Guide

  1. Put 1 space before the {
  2. Put 2 spaces before property names
  3. Put 1 space before property values
  4. Put a ; after property values
  5. Put each property on a new line
  6. Put each selector on a new line
  7. Use single quotes around property values when necessary
  8. A blank line separates each CSS rule
  9. Put properties in alphabetical order

CSS Style Guide

Use shorthand properties where possible
